Code Submissions

Challenge Styles

Least Lines Of Code

Limited file size submission. Original program unrestricted size. (Submitted program kb size) < $kbLimit. i.e. Original program occupies 15 kb. Goal is 10kb or less. Submitted program occupies 8 kb.

Standard Example

function helloworld() {} console.log(":)")

Submission Example

function helloworld() {} \n console.log(":)")

Restricted Operators/func

Limited operator selection allowed. Original program contains zero operator restrictions. Submitted program operators restricted to any n operators. i.e. - Submitted program may only use 3 operators. - Submitted program may only use (+, <<, ||) - Submitted program may not use (+,-,/,*)

Standard Example

function helloworld() {} console.log(":)")

Submission Example

function helloworld() {} \n console.log(":)")

Restricted Data Types

Limited data types allowed. Original program contains zero restrictions. Submitted program maybe restricted to any n datatypes. i.e. - Submitted program may only use 2 datatypes. - Submitted program may only use (booleans, and strings) - Submitted program may not use (numbers)

Standard Example

function helloworld() {} console.log(":)")

Submission Example

function helloworld() {} \n console.log(":)")

Restricted RAM capacity

Limited RAM exceeds. Original program contains zero restrictions. Submitted program maybe restricted to 100mb max RAM. i.e. - Submitted program max RAM 300mb. - Submitted program max RAM 10mb.

Standard Example

function helloworld() {} console.log(":)")

Submission Example

function helloworld() {} \n console.log(":)")

Restricted Memory Declarations

Limited memory declarations. Original program contains zero restrictions. Submitted program contains n max memory declarations. i.e. - Submitted program may only use less than 4 memory declarations

Standard Example

function helloworld() {} console.log(":)")

Submission Example

function helloworld() {} \n console.log(":)")
